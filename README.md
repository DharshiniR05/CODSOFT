## 📂 Tasks Completed

---

### 🚢 Task 1: Titanic Survival Prediction

**Problem:** Predict whether a passenger survived the Titanic disaster.

**Approach:**

* Handled missing values
* Converted categorical data into numerical form
* Used Logistic Regression model

**Result:**
Achieved approximately **80% accuracy** in predicting survival.

---

### 🎬 Task 2: Movie Rating Prediction

**Problem:** Predict movie ratings based on features like genre, director, actors, duration, and votes.

**Approach:**

* Cleaned and preprocessed dataset
* Converted categorical data using Label Encoding
* Used Linear Regression model

**Result:**
Model predicts movie ratings with reasonable accuracy.

---

### 🌸 Task 3: Iris Flower Classification

**Problem:** Classify iris flowers into three species.

**Species:**

* Setosa
* Versicolor
* Virginica

**Approach:**

* Encoded species labels
* Used Decision Tree Classifier

**Result:**
Achieved **95%–100% accuracy**.

---


### 💳 Task 5: Credit Card Fraud Detection

**Problem:** Detect fraudulent credit card transactions.

**Approach:**

* Handled imbalanced dataset
* Used Logistic Regression model
* Evaluated using confusion matrix and classification report

**Result:**
Achieved **~99% accuracy** in fraud detection.
